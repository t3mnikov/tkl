# tkl

`tkl` (*traslit-keyboard-layout*) — маленькая CLI-утилита на Go, которая конвертирует текст, случайно набранный не в той раскладке клавиатуры, в правильный вариант.

Классический случай: хотели написать `wednesday`, а получили `цудсщыфн` — `tkl` это исправит.

## Возможности

- Конвертация `RU → EN` и `EN → RU`.
- Корректная обработка верхнего/нижнего регистра.
- Поддержка знаков препинания и пунктуации ЙЦУКЕН-раскладки (`ё`, `Ё`, `№`, `"` и т. п. в пределах таблицы).
- Символы, которых нет в таблице (цифры, пробелы, другие), остаются без изменений.

## Установка

Сборка из исходников:

```bash
git clone https://github.com/t3mnikov/tkl.git
cd tkl
go build -o bin/tkl ./cmd/tkl
```

Сборка релизного бинарника под Linux через `Makefile`:

```bash
make build_linux
# результат: bin/linux/tkl
```

## Использование

```text
tkl <en|ru> <text>
```

- Первый аргумент — **целевая** раскладка, в которой должен оказаться результат.
- Далее — исходный текст. Его можно передавать одной строкой (в кавычках) либо несколькими аргументами — они будут склеены через пробел.

## Примеры

Текст набрали в EN, а хотели в RU:

```bash
$ tkl ru ghbdtn
привет

$ tkl ru "Ghbdtn, vbh!"
Привет, мир!
```

Текст набрали в RU, а хотели в EN:

```bash
$ tkl en цудсщьу
wednesday

$ tkl en "Руддщ, цщкдв!"
Hello, world!
```

Несколько аргументов склеиваются автоматически:

```bash
$ tkl en цудсщьу ghbdtn
wednesday ghbdtn
```

> Символы, которые отсутствуют в таблице соответствия (цифры, пробелы и пр.), копируются в вывод как есть.
